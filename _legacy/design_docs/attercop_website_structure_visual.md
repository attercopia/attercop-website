# Attercop Website Content Structure & Brand Guidelines v3
## Graphic-Driven Enterprise Design Approach

## Executive Summary
This document outlines a visual-first website structure for Attercop, adopting a sophisticated graphic-driven approach similar to PwC's design language. The structure emphasises visual storytelling, data visualisations, interactive elements, and bold typography whilst maintaining enterprise credibility.

---

## Visual Design Philosophy

### Core Design Principles

#### Visual Hierarchy Through Graphics
- **Hero Visuals**: Full-width animated graphics, not stock photos
- **Data Visualisations**: Live, interactive charts and graphs
- **Iconography**: Custom icon system for concepts
- **Infographics**: Complex ideas simplified visually
- **Motion**: Subtle animations that explain concepts

#### Grid-Based Card Layout
- **Modular Design**: Content in digestible cards
- **Visual Entry Points**: Each card has graphic element
- **Progressive Disclosure**: Hover states reveal depth
- **Asymmetric Balance**: Varied card sizes for dynamism

#### Typography as Design Element
- **Large Type**: Headlines as visual elements (72-96px)
- **Mixed Weights**: Thin/Bold contrasts
- **Vertical Type**: For visual interest
- **Number Graphics**: Statistics as hero elements

- **Hero Headlines**: Sora Display, 72-120px

### Special Treatments

- **Variable Fonts**: Weight animations

## Animation Guidelines

### Performance Principles
- GPU-accelerated transforms only
- Lazy loading for below fold
- Reduced motion media query respect
- 60fps target for all animations

### Animation Types
- **Entrance**: Fade up, scale in
- **Hover**: Smooth transitions
- **Scroll**: Parallax, reveal
- **Interactive**: Physics-based
- **Loading**: Skeleton screens

## Content Strategy for Visual Design

### Less Text, More Visual
- Headlines: Maximum 10 words
- Subheads: Maximum 20 words
- Body: Hidden until interaction
- CTAs: Action-oriented, 2-4 words

### Visual Hierarchy
- Graphics draw attention
- Headlines provide context
- Details on demand
- CTAs clear and prominent
